Sailthru - 2021 Personalization Index

PHP Site using a Wordpress CMS

Role: Freelance Web/Front-End Developer

Codebase: PHP, WordPress, HTML, CSS, and responsive design

Duration: February 2021 - March 2021

Launched: March 2021

Live Site

As a Freelance Web/Front-End Developer for Sailthru's 2021 Personalization Index, I developed dynamic pages in PHP and using Wordpress as a CMS and I performed QA on the site.

Overview

This site was to have three main pages, and then 100 pages with the same layout but different content. These pages needed to be built out so that there were no design errors, no matter how much text or what shape each company logo was. It needed to work on all device sizes, and the non-technical team needed to be able to input their text and images without using a developer.

Approach

The site was built statically with dummy text using PHP. Different amounts of text and different sized logos were added to troubleshoot any CSS bugs. Once all pages were flawless and ready to go live, I translated the PHP code to accept dynamically added text and images, added through the WordPress page editors.
I then performed QA on any bugs found by our team or the Sailthru team.

Results

The site was launched and works with all logo sizes and amount of text. The clients were very happy with the results and the ease of use for them.
