What Can I Make?

React application using third party API (Spoonacular)

Codebase: React, dotenv, CSS, responsive design, and media queries.

I planned and coded this web app before I had ever worked with a team or client. The goal was to create an app that would return recipes to the user based on dietary preferences, the ingredients they have on hand, and ingredients they do not want to use.

Created between February and May 2020, Version 01 launched May 2020
